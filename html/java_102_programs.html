  • 사용할 수 있는 언어의 기능, 변수, 숫자는 다음과 같다.
    • 기능 while
    • 변수 n i
    • 숫자 1 2

템플릿 코드

class Main
{
  public static void main(String[] args)
  {
    var n = 2;
    var i = 0;
    while (i < n)
    {
      System.out.println(0);
      i = i + 1;
    }
    System.out.println(i);
  }
}

실행 결과

0
0
2

프로그램 설명

프로그램은 위의 소스 코드에서 public static void main(String[] args) { ... } 코드의 { ... } 사이에 있는 코드들을 첫 번째 줄부터 마지막 줄까지 한 줄씩 차례대로 실행한다. 줄 단위로 실행 과정을 살펴보자. 살펴볼 코드들만 떼어보면 다음과 같다.

    var n = 2;
    var i = 0;
    while (i < n)
    {
      System.out.println(0);
      i = i + 1;
    }
    System.out.println(i);
  • 1 번째 줄의 var n = 2; 코드는 변수 목록에 n이라는 이름의 변수를 등록하고 값을 2로 설정한다. 변수 목록은 다음과 같다.
변수명 값
n 2
  • 2 번째 줄의 var i = 0; 코드는 변수 목록에 i라는 이름의 변수를 등록하고 값을 0으로 설정한다. 변수 목록은 다음과 같다.
변수명 값
n 2
i 0

  • 3 번째 줄의 while (i < n) { ... } 코드는 i < n 식을 계산해 참 거짓을 판단한다. i < n 식은 i가 n보다 작다면이라는 의미다. 위의 변수 목록에서 i의 현재 값은 0이고, n은 현재 값은 2다. 0은 2보다 작으므로 참이다. 참이면 { ... } 사이의 코드들을 순서대로 실행한다.

  • 4 번째 줄의 { 기호는 코드 묶음의 시작 기호일 뿐이므로 넘어간다.

  • 5 번째 줄의 System.out.println(0); 코드는 화면에 숫자 0을 출력하고 줄바꿈한다. 출력 결과는 다음과 같다.

0

코딩에서는 숫자 하나도 식이다. 숫자를 적을 수 있는 곳에는 식을 작성할 수 있다. 변수 또한 숫자와 같으므로 숫자를 적을 수 있는 곳에는 변수도 사용할 수 있다.

  • 6 번째 줄의 i = i + 1; 코드는 = 기호 오른쪽의 i + 1 식을 계산해 변수 i의 값을 변경한다. 변수 목록에서 i의 현재 값이 0이므로 i = 0 + 1;과 같다. 식을 계산하면 i = 1;이 되므로, 변수 i의 값을 1로 수정한다.
이름 값
n 2
i 1
  • 7 번째 줄의 } 기호는 코드 묶음의 종료 기호일 뿐이므로 넘어간다. 그런데 코드 묶음의 시작이 while 이었으므로, 다시 3 번째 줄로 이동해 실행을 계속한다.

while이 ~ 하는 동안 이라는 뜻의 단어인 것처럼, while (식) { ... } 코드에서 { ... } 사이의 코드들은 (식)의 결과가 참인 동안 반복해서 실행된다.

  • 3 번째 줄의 while (i < n) { ... } 코드에서, 변수 i의 현재 값은 1이고, 변수 n은 현재 값은 2다. 1은 2보다 작으므로 참이다. 따라서 다시 { ... } 사이의 코드들을 순서대로 실행한다.

  • 4 번째 줄의 { 기호는 코드 묶음의 시작 기호일 뿐이므로 넘어간다.

  • 5 번째 줄의 System.out.println(0); 코드는 화면에 숫자 0을 출력하고 줄바꿈한다. 출력 결과는 다음과 같다.

0
0
  • 6 번째 줄의 i = i + 1; 코드에서, 변수 i의 현재 값이 1이므로 i = 1 + 1;을 실행해 변수 i의 값을 2로 수정한다.
이름 값
n 2
i 2

  • 7 번째 줄의 } 기호는 코드 묶음의 종료 기호일 뿐이므로 넘어간다. 그런데 코드 묶음의 시작이 while 이었으므로, 또 다시 3 번째 줄로 이동해 실행을 계속한다.

  • 3 번째 줄의 while (i < n) { ... } 코드에서, 변수 i의 현재 값은 2고, 변수 n은 현재 값은 2다. 2는 2보다 작지 않으므로 거짓이다. 따라서 { ... } 사이의 코드들을 실행하지 않고, 8 번째 줄의 코드로 이동한다.

  • 8 번째 줄의 System.out.println(i); 코드는 화면에 변수 i의 값을 출력하고 줄바꿈한다. 변수 목록에서 i의 현재 값은 2이므로 출력 결과는 다음과 같다.

0
0
2

그 다음 줄의 } 기호는 public static void main(String[] args) { ... } 코드의 코드 묶음 종료 기호이므로 프로그램을 종료하며 실행이 끝난다.
